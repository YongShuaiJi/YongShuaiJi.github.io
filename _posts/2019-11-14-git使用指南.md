获取Git仓库:

创建本地仓库，Git创建一个头指针

$ git init

该命令将创建一个名为.git 的子目录，这个子目录含有你的初始化的Git仓库中所有的必须文件，这些文件都是Git仓库骨干。 但是，在这个时候，我们仅仅是做了一个初始化的操作，你的项目里的文件还没有被跟踪。
